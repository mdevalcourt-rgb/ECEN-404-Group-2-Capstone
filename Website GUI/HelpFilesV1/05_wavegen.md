# Waveform Generator

Use the waveform generator to output AC signals into your circuit. This replaces the AD2's Wavegen tool and the function generator in MultiSim simulations.

## Pins Used

| Pin | Location | Function |
|---|---|---|
| **Output** | Top header, pin 3 | Signal output |
| **GND** | Top header, pins 1–2 | Signal return / ground reference |

The output is generated by the AD9833 DDS chip over SPI, capable of sine, triangle, and square waves up to 200 kHz.

## Wiring

1. Connect the **Output** pin to the input node of your circuit (where the Lab Manual shows V\_in or the function generator).
2. Connect one of the adjacent **GND** pins to your breadboard ground rail.

## Settings

Configure these parameters on this screen before enabling the output:

| Parameter | Description |
|---|---|
| **Waveform** | Sine, Square, or Triangle |
| **Frequency** | In Hz — set to the value specified by your lab |
| **Amplitude** | Peak voltage of the signal |
| **Phase** | Phase shift in degrees (0–360°) |

> **Note:** The waveform generator **does not support DC offset**. All waveforms are centered around 0 V. If a lab calls for a DC offset, you will need to add it externally with a bias circuit.

## Common Lab Settings

| Lab | Waveform | Frequency | Amplitude | Notes |
|---|---|---|---|---|
| RC/RL Transients | Square | 20 Hz | 1 V | Lab calls for 1 V offset — requires external bias |
| AC Signals | Sine | 30 Hz | 2 V | Set phase to 50° |
| Op Amp | Sine | 30 Hz | 0.5 V | — |
| Op Amp Integrator | Sine / Square / Triangle | 30 Hz | 0.5 V | Switch waveform type between tests |
| Frequency Response | Sine | 1–1200 Hz | 2 V | Sweep through frequency table |

## Tips

- The Output pin produces the signal relative to the adjacent GND pins. Always use these GND pins — do not rely solely on a GND connection elsewhere on the MADAM3.
- When sweeping frequencies (Lab 6), you may need to adjust the oscilloscope timebase after each frequency change.
